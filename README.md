# HellNRock Firefox Theme

A premium custom theme for **Firefox ONE**.

![New Tab Preview](https://hellnrock.com/wp-content/uploads/Image_qx9friqx9friqx9f.png)

### 🎨 Color Palette
- **Deep Black** — `#101212`
- **Ancient Gold** — `#C7B182`
- **Main Purple** — `#634F5F`
- **Cream** — `#E8E3DE`

### ✨ Features
- Stunning New Tab background with elegant overlay
- Smooth flowing gold animation on active tab (Canva-inspired)
- Full deep black UI across the entire browser
- Vertical left sidebar with hover expansion
- Premium glows and glassmorphism effects

### 📥 Installation

1. Download this repository.
2. Copy the files to your Firefox `chrome` folder:
   - **Windows**: `%APPDATA%\Mozilla\Firefox\Profiles\XXXXXXX.default-release\chrome\`
3. Restart Firefox completely (close all windows).
4. (Optional) Enable Bookmarks Toolbar:
   - Go to `about:config` → set `browser.toolbars.bookmarks.visibility` to `always`
